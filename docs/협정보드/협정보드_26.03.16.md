# 협정보드 작업 메모 (2026-03-16)

## 1. 개찰일 입력 버그 수정

- 대상: 협정보드 헤더 `개찰일`
- 문제:
  - 분 입력 시 `30`이 `03`처럼 꼬이는 현상
  - 분을 지우면 날짜/오전오후/시간까지 같이 초기화되는 현상
- 원인:
  - 시/분 입력 중에도 전체 `bidDeadline`를 즉시 재조립
  - 재조립된 값을 다시 파싱해 입력칸 전체를 덮어쓰는 구조
- 조치:
  - 시/분 입력 중에는 로컬 상태만 유지
  - blur 시점에만 유효한 값으로 최종 반영

## 2. LH 100억~300억 엑셀 내보내기 구현

- 템플릿 파일:
  - `템플릿/LH간이종심100-300억_템플릿.xlsx`
- 템플릿 키:
  - `lh-100to300`
- 헤더 셀 매핑:
  - 추정가격: `E1`
  - 기초금액: `E2`
  - 시공비율기준금액: `H1`
  - 공고명/공고번호: `O1`
  - 개찰일: `P2`
  - 의무지역/의무지분: `U2`
- 본문 배치:
  - LH 50억 미만 / 50억~100억과 동일하게 `C5/D5`, `C7/D7` 패턴 사용
  - 지분, 경영상태, 실적액 위치 동일

## 3. LH 100억~300억 엑셀 내보내기 후속 조정

- 품질점수 입력 위치 수정
  - 기존 LH 구간처럼 지분 아래가 아니라 경영상태 아래로 이동
  - 대표사 기준 `P6`, 구성원1 `Q6` 형태로 입력
- 경영점수 만점 기준 수정
  - LH 100억~300억은 8점 만점 기준
  - 엑셀 내보내기 시 8점 만점이면 경고색이 뜨지 않도록 수정
- 의무지역/의무지분 표시 단순화
  - 예: `경기의무30%, 5개사` 대신 `경기30%`
- 품질 만점 셀 미입력
  - 템플릿 수식 셀(`AD` 계열)은 직접 덮지 않도록 수정
- 품질 점수 강조
  - 기본값 초과 품질점수만 노란색 셀 처리
- 업체명 셀 표시 보강
  - 가능지분 표시
  - LH 100억~300억에서 품질 기본값 초과 시만 `품질 xx.xx` 문구 표시

## 4. 협정보드 테이블 폭 조정

- 목적:
  - 접기/펼치기와 별개로 펼친 상태에서도 최대한 한눈에 보이게 조정
- 주요 조정 대상:
  - 시공실적
  - 지분
  - 경영상태
  - 요약칸(실적총점, 품질점수총점, 지역경제기여도합 등)
- 추가 조정:
  - 승인 컬럼 기본값은 다시 접힘 상태로 설정
  - 기본 펼침 상태는 `신인도/경영상태/시공실적/시평액` 기준으로 정리

## 5. LH 100억~300억 테이블 구조 조정

- 지역경제기여도 개별 슬롯 칸 제거
  - 합계 칸은 유지하고, 아래 줄에 지역업체 지분 합만 표시
- 지역경제기여도 합 칸 표시
  - 위: 점수
  - 아래: 지역업체 지분 합
- 실적계수 칸 추가
  - `실적(11점)` 옆에 별도 열 추가
  - 소수점 4자리 표시
  - 볼드체 처리
- 주의:
  - 중간에 `colgroup` 순서와 헤더/본문 순서가 불일치해서
    `실적계수` / `실적총점` 폭 조정이 반대로 먹는 문제 있었음
  - 최종적으로 `colgroup` 순서를 헤더/본문 순서와 일치하도록 수정

## 6. 품질총점 표시 방식 확인 및 정리

- 쟁점:
  - LH 100억~300억 품질총점이 앱에서는 `92.035`, 엑셀에서는 `92.04`처럼 보이는 차이
- 확인 결과:
  - 앱 계산 자체는 합산 후 표시 단계에서 `toFixed(2)` 반올림
  - 엑셀도 표시 형식상 소수 둘째 자리 반올림으로 보이는 것으로 확인
- 최종 조치:
  - 앱 표시도 엑셀과 동일하게 소수 둘째 자리 반올림 기준 유지

## 7. 셀 내부 줄바꿈 처리

- 시공실적:
  - 좁은 칸에서도 줄바꿈되도록 처리
  - 폰트 크기는 유지하고 칸 폭/패딩만 조정
- 시평액:
  - 시공실적과 동일하게 줄바꿈 가능하도록 처리
- 경영 총점:
  - 총점 셀 값이 넘칠 경우 줄바꿈되도록 처리
- 주의:
  - 업체별 경영점수는 한때 줄바꿈 처리했으나 의도와 달라 원복

## 8. 업체별 경영점수 색상 규칙 추가

- 기존:
  - 만점 미달 시 빨간색 경고
- 추가:
  - 업체별 경영점수도 발주처/금액대별 만점 기준 도달 시 초록색 볼드 처리
- 기준:
  - 이미 사용 중인 `perSlotMax` 기준 재사용
  - 예:
    - 행안부 15점 만점 구간은 15점일 때 초록색
    - LH 100억~300억은 8점일 때 초록색

## 9. LH 50억 미만 / 50억~100억 폭 조정

- LH 50억 미만:
  - 시공경험점수 폭 축소
  - 순공사원가가점 폭 축소
- LH 50억~100억:
  - 시공경험점수 폭 축소
  - 하도급 폭 축소
  - 자재 폭 축소
  - 순공사원가가점 폭 축소

## 10. 주의 메모

- 오늘 작업 중 `실적계수`와 `실적총점` 폭 조정은 여러 차례 재조정됨
- 원인은 변수명 문제가 아니라 `colgroup` 순서 불일치였음
- 이후 같은 종류의 칼럼 폭 수정 시:
  - 헤더 순서
  - 본문 렌더 순서
  - `colgroup` 순서
  - `COLUMN_WIDTHS`
  이 4개를 같이 확인해야 함
